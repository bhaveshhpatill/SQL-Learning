# SQL Day 1

## Topics Learned

- What is SQL?
- What is a Database?
- What is a Table?
- Rows vs Columns
- Introduction to MySQL Workbench

## Key Learnings

- SQL is the language used to communicate with a database.
- A database stores company data in an organized way.
- Tables store one type of information.
- Columns describe the type of information.
- Rows represent one complete record.
- MySQL Workbench is the application used to write SQL queries.

## Business Problem

Understanding where company data is stored and how Business Analysts retrieve it.

## Reflection

Today I learned that SQL itself doesn't store data. SQL is the language used to ask questions to a database.
